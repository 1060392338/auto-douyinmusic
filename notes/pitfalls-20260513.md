# 2026-05-13 批量导出踩坑记录

## 核心发现

### dispatchEvent 返回 None ≠ 失败 🔥
JS `dispatchEvent()` 返回 `undefined` → Python `P.run_js()` → `None`。
**不能用 `if r == 'clicked'` 判断成败。** 用 URL 变化/弹窗出现判断，或在 IIFE 中显式 `return 'ok'`。

### 素材列表渲染需 12+4 秒
导航到 `/studio/create` 后至少等 **12秒**，滚动到底部再等 **4秒**，`titleRow` 才出现。

### heredoc 后台输出不可见
`terminal(background=true)` + heredoc Python 脚本输出完全消失。
用文件脚本 + 前台 `terminal()` 运行。

### P.ele 超时
`P.ele("tag:body")` 在页面加载中可能超时 → 用 `P.run_js("return document.body.textContent")`。

## 已验证导出流程（一次成功率 100%）

```
AI编辑 → 去编辑器 → 等15s → 点导出 → 改项目名 → 确认 → 等跳转
```

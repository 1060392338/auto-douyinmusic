# 登录操作手册

> 推荐方式：验证码登录（无需扫码，无需截图）

---

## 选择器

| 元素 | 选择器 | 说明 |
|------|--------|------|
| 登录按钮 | 页面右上角「登录」 | 未登录时可见 |
| 验证码登录tab | 弹窗中「验证码登录」 | 在二维码登录旁边 |
| 手机号输入 | `input[placeholder*="手机号"]` | |
| 获取验证码 | `button:has-text("获取验证码")` | 点击后等短信 |
| 验证码输入 | `input[placeholder*="验证码"]` | 用户提供 |
| 登录提交 | `button:has-text("登录")` | |

## 流程

1. 确认 Chrome 已启动（9223）
2. 导航到 `https://music.douyin.com/studio/create`
3. 若已跳转到资产页（已登录）→ ✅ 跳过
4. 否则点「登录」按钮
5. 切到「验证码登录」tab
6. 输入手机号 → 「获取验证码」
7. 用户提供验证码 → 输入 → 点登录
8. Session 自动持久化到 `chrome_data/`
9. 验证：导航 `/studio/create` 进入作曲页

## 注意

- 网页版抖音短逻辑：验证码 60s 有效
- 登录态持久化到 `chrome_data/`，下次重启 Chrome 仍有效
- session TTL ~5-10 分钟不活跃后过期 → 需要重登
- 重新登录后：删除旧凭据 → 存新凭据 → 同步 skills + shared-memory
